# Note Tidy

整理纯文本（.txt）笔记，让内容变得整洁、清晰、易读。

## 核心原则

- **保持原意**：不增删用户的原内容，只调整格式
- **简洁清爽**：不做过度修饰，保持纯文本风格
- **可读性优先**：让笔记一目了然
- **智能识别**：自动判断内容的类型（列表、段落、代码、数据等），选择合适的整理方式

## 智能识别与分类

读取内容后，先判断它属于哪种类型，再针对性处理：

| 内容特征 | 判断为 | 处理方式 |
|---|---|---|
| 大量逗号/分隔符分隔的项目 | **列表类** | 逐行排列，自动去重，可选排序 |
| 有标题/章节结构 | **文档类** | 统一标题层级，调整段落间距 |
| 纯连续文字 | **段落类** | 合并断行，清理空格，规范标点 |
| 表格/对齐数据 | **数据类** | 保持对齐，统一列宽 |

## 整理规则

### 1. 标题层级统一
- 一级标题：`= 标题 =`（上下各空一行）
- 二级标题：`== 标题 ==`（上下各空一行）
- 三级标题：`=== 标题 ===`
- 如果原文用了其他格式（如 `#`、`**` 等），统一转换为上述格式

### 2. 段落间距
- 段落之间保留**一个空行**
- 同一段落内不要有多余换行
- 连续的空行（超过1行）压缩为1行

### 3. 多余空格清理
- 行首多余空格 → 删除
- 行尾多余空格 → 删除
- 连续多个空格 → 保留（可能是对齐用的缩进）

### 4. 列表类处理（重要）
- 如果原文是用逗号、顿号、分号分隔的一长串 → 转换为**逐行排列**
- 无序列表统一用 `- ` 开头
- 有序列表保持原数字顺序
- 自动检测并**删除完全重复的条目**
- 列表项之间不空行

### 5. 分隔线
- 如原文有分隔线，统一为 `---`（上下各空一行）

### 6. 输出摘要（增强功能）
整理完成后，在末尾添加一行简洁的统计摘要，格式：
```
---
📋 共 X 项 | 去重 X 项 | 分类：xxx
```

## 不做的事
- ❌ 不改动原文实质内容（除非明显的拼写错误）
- ❌ 不加目录
- ❌ 不加标签
- ❌ 不转 Markdown 格式
- ❌ 不添加原文没有的标题
