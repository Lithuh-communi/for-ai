# 📱 手机 FTP 文件操作 — AI Skill 使用说明

## 这是什么？

这是一个 **AI Skill（技能文档）**，给其他 AI 助手（如 Claude、ChatGPT 等）阅读后，它们就能通过 FTP 连接你的安卓手机，直接操作手机上的文件。

## 📂 文件清单

| 文件 | 说明 |
|------|------|
| `phone-ftp-access_SKILL.md` | **核心 Skill 文档** — 给 AI 读的 |
| `index.html` | AI 工作台导航页（手机浏览器打开） |

## 🚀 怎么用（给其他 AI）

### 方法一：直接粘贴内容（最快）

把 `phone-ftp-access_SKILL.md` 的**全部内容**复制，发给 AI 说：

> 「你先读一下这个 Skill 文档，然后就能连我手机操作文件了」

### 方法二：上传文件

如果 AI 支持上传文件，直接把 `phone-ftp-access_SKILL.md` 传给它。

## 🔌 使用前准备

每次让 AI 连手机前，你需要：

1. 打开 **Material Files** App
2. 点右上角三个点 → **FTP 服务器** → **启动**
3. 把屏幕上显示的 FTP 地址发给 AI（如 `ftp://192.168.x.x:2121/`）

> 💡 手机 IP 可能随 WiFi 变化，每次都要给 AI 最新的地址哦

## ✅ AI 能帮你做的事

| 操作 | 举例说明 |
|------|---------|
| 📂 浏览文件 | "看看我手机 Download 里有什么" |
| 📥 下载到工作区 | "把 DCIM 里的照片下载到工作区" |
| 📤 上传到手机 | "把这个文件存到我手机里" |
| 🗑 删除整理 | "删掉 Download 里的这个文件" |
| 📦 批量操作 | "把所有 apk 列出来" |
| 🔄 中转处理 | "把这张图片压缩后再存回手机" |

## ⚠️ 注意事项

- 手机锁屏后 FTP 可能会断，需要重新启动
- 连接成功后尽快操作，避免超时
- 只能访问手机内部存储（不能访问系统文件）
