# 📢 更新总结 — 2026-06-20

## 🆕 新增内容

### 1. ADB over WiFi 远程控制
- 通过「无线调试」配对连接手机
- 可执行任意 shell 命令
- 无需 root，Android 11+ 通用

### 2. OpenCV 计算机视觉
- 版本: 4.6.0
- 支持截图分析和图像处理
- 可标注识别结果

### 3. TesserOCR 文字识别
- 中文简体 + 英文语言包
- 可识别屏幕上所有文字位置
- 支持按文字查找按钮坐标

### 4. smart_tap.py 智能点击器
- 一键截图 + OCR 识别 + 精准点击
- 示例: `python3 smart_tap.py 发送`

## 🚀 使用方法

```bash
# 连接
adb connect 10.150.0.1:40745

# 截屏
adb exec-out screencap -p > /tmp/screen.png

# OCR 识别文字
python3 smart_tap.py

# 找按钮并点击
python3 smart_tap.py 发送
```

- 中文输入用 `input text` 会报错，需用剪贴板方式替代
- 游戏类 OpenGL 渲染的按钮 OCR 可能认不全
